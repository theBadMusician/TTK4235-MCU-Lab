# Lab 3 Oppgave 1 - GPIO

## Opgave 1.2

If the shield is not present, the buttons are connected directly to pins labeled BUTTON[1-4]. They are mapped to the GPIO pins on the target MCU specified as P0.1[3-6].

The buttons are active low, meaning that when a button is pressed, it connects the corresponding GPIO pin to ground (GND), resulting in a low signal.
